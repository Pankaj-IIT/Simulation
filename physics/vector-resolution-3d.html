Vector V (Yellow)

3D Vector Resolution (Direction Angles):

Vx = |V|·cos(α) — component along X
Vy = |V|·cos(β) — component along Y
Vz = |V|·cos(γ) — component along Z

|V| = √(Vx² + Vy² + Vz²)

cos²(α) + cos²(β) + cos²(γ) = 1

α, β, γ = direction angles
(angles between V and each axis)
