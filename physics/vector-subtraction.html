Controls

Vector A (Green)
Vector B (Orange)
Difference D = A - B (Yellow)
Vector Subtraction:

D = A - B = A + (-B)

-B has same magnitude as B but opposite direction
Dx = Ax - Bx
Dy = Ay - By

|D| = √(Dx² + Dy²)

Geometrically: D connects tip of B to tip of A
(when both start from same origin)
